Rotate the point with coordinate ‘cord’ with an anticlokwise angle ‘angles’ around the axes ‘axes’. The function return the new rotated coordinates of the points.

Parameters:
  • cord – np.array (Nx3) or list with (x,y,z)
  • angles – List of (anticlockwise) angle of rotation.
  • axes – List of axes of rotation.
  • unpacked – If True, coord are on the form (x,y,z), if False cord are a Nx3 array with x in the 0-col, y 1-col and z 2-col
  • unpack – If True return the new cord in the form (xn,yn,zn), if False return the coord in a Nx3 array with x in the 0-col, y 1-col and z 2-col
  • reference – rh if the system is right-handed, lh if the system is left-handed.
  • xoff – X-offset after the rotation
  • yoff – Y-offset after the rotation
  • zoff – Z-offset after the rotation
Returns:
